La Caja Discursiva

Es una obra interactiva que se presento el 14 de Diciembre de 2019, dentro del marco de “Artimañas: Festival de Arte Interactivo Vol. 7”. El festival tiene como objetivo presentar los trabajos finales de los alumnos de la carrera Diseño Multimedial de la Universidad Nacional de La Plata.

Tu Peso en Agua

Programación de obra interactiva realizada dentro del LAIT para el evento "Planeta Agua", que se presentó en el Centro de Arte de la UNLP (2019).

Battle City Worlds

Videojuego multijugador de exploración y acción. Inspirado en el clásico Battle City. Actualmente se encuentra en desarrollo. Próximamente estaré liberando el código bajo la licencia MIT :).

DC Motor + App

Control de un Motor DC mediante pulsadores o desde un app para smartphone. Este desarrollo lo hicimos en Moduchip bajo encargo junto con mi socio Dardo Fortunato.

ESP32 + API Rest + Flutter

Sistema IoT de monitoreo y control de temperatura. Para este proyecto se utilizó una ESP32 como microcontrolador, se desarrolló una API Rest, y se programó una aplicación para smartphone en Flutter.

COD05

COD05 (2014-2018) es un sistema de partículas virtuales, capaces de interactuar con las personas. Mediante una cámara Kinect se captura el movimiento de las personas, y esto afecta al sistema de partículas.

Uturunku

Obra interactiva del Estudio Biopus, donde participé como ayudante técnico. La obra fue presentada en el Centro Cultural de San Martín (2017).

Diseño de PCB

Uno de mis diseños de PCB realizado en EasyEda.

Escaneando cuerpo con Kinect

Desarrollo de un programa en Processing que utiliza la cámara Kinect para exportar un modelo 3D en formato OBJ. De tal manera, el modelo se puede abrir en cualquier programa de 3D como puede ser Blender.

Comisión Provincial por la Memoria

Obra interactiva realizada para la Comisión Provincial por la Memoria (2019).

Inside of the Code

Inside of the Code (2016) es una obra experimental y conceptual realizada con Processing. El código del programa tiene las instrucciones de leer su propio código fuente para mostrarlo en un espacio recorrible y en 3D. Es una obra conceptual, que invita a ver el “resultado” y el “cómo se hizo” al mismo tiempo.

Mapping Interactivo para Samsung

Participé como asistente técnico en el Estudio Biopus, para la realización de un mapping interactivo para Samsung.

La Escala del Universo

La Escala del Universo (2015) es una obra interactiva desarrollada para el Centro Cultural de la Ciencia (C3). Participé en el desarrollo del software.

Piano - Escuela de Verano (2017)

Piano es un software didáctico, pensado para la enseñanza sobre la música sintética. Fue desarrollado para la Escuela de Verano de la Provincia de Buenos Aires (2017). Con este software los alumnos pueden crear visuales al mismo tiempo que componen melodías. Además, pueden cambiar la forma de onda (cuadrada, diente de sierra, sinusoide, etc.) para escuchar los distintos timbres.

Mandalasoft (2015)

Es un software para la creación de mandalas. Las mandalas son construidas mediante la utilización de primitivas como círculos y anillos. Además, utiliza la lógica de capas, brindando un gran potencial para crear mandalas muy variadas.
